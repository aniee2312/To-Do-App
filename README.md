# Todo App

## Project Overview
This Todo App helps users add, manage, update, and organize their daily tasks efficiently. Users can create tasks and manage them easily with CRUD operations.

## Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Update task details
- Store tasks using Local Storage
- Data remains available after refreshing the page
- User-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript
- React.js
- Local Storage

## CRUD Operations Used
This project uses CRUD operations:

- Create → Add new tasks
- Read → Display tasks
- Update → Edit tasks
- Delete → Remove tasks
